# Doptor Super App

## Structure

- **frontend/**: Frontend applications (Web, Mobile) and shared UI.
- **backend/**: Backend services (API, Workers, Realtime) and database schemas.
- **infra/**: Infrastructure configuration (Docker, K8s, Terraform).

## Getting Started

### Prerequisites

- Node.js 20+
- Docker (for database)

### Installation

```bash
npm install
```

### Running Locally

```bash
npm run dev
```

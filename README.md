# Doptor Super App

## Structure

- **frontend/**: Frontend applications (Web, Mobile) and shared UI.
- **backend/**: Backend services (API, Workers, Realtime) and database schemas.

## Getting Started

### Prerequisites

- Node.js 20+
- Docker (for database)

### Installation

```bash
pnpm install
```

### Running Locally

```bash
pnpm dev
```
